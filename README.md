# Python PDF Merger

A desktop application built with Python, Tkinter, and pypdf to merge multiple PDF files into a single PDF document.

## Features

- Merge multiple PDF files
- Simple graphical interface
- Custom output filename
- Remove and clear selected files

## Requirements

- Python 3.x
- pypdf

## Run

```bash
pip install pypdf
python pdf_merger.py
```
